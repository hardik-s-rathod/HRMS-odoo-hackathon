# HRMS - Human Resource Management System

## About This Project

HRMS is a comprehensive Human Resource Management System built with Laravel. The system provides features for employee management, attendance tracking, leave management, payroll processing, and administrative controls.

### Key Features

-   **Employee Management** - Manage employee profiles and information
-   **Attendance Tracking** - Daily and weekly attendance records
-   **Leave Management** - Apply, approve, and manage employee leave requests
-   **Payroll Management** - Process payroll and generate reports
-   **Role-Based Access** - Admin and Employee role permissions
-   **Dashboard** - Personalized dashboards for admins and employees
-   **Authentication** - Secure login system with role-based access control

### Tech Stack

-   **Framework**: Laravel (PHP)
-   **Database**: MySQL
-   **Frontend**: Bootstrap & Blade Templates
-   **Authentication**: Laravel UI
